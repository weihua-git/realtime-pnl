# 🚀 Web 配置界面 - 5 分钟快速开始

## 第一步：安装依赖（1 分钟）

```bash
npm install
```

## 第二步：启动服务（1 分钟）

### 方式一：一键启动（推荐）

```bash
./start-all.sh
```

### 方式二：分别启动

```bash
# 终端 1：启动 Web 配置界面
npm run web

# 终端 2：启动监控程序
npm start
```

## 第三步：访问配置页面（1 分钟）

### 电脑访问

打开浏览器，访问：http://localhost:3000

### 手机访问

1. 确保手机和电脑在同一局域网
2. 查看电脑 IP 地址：
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   
   # 或
   ip addr show
   ```
3. 手机浏览器访问：http://你的IP:3000

## 第四步：修改配置（2 分钟）

### 示例：修改 ETH 价格目标

1. **找到"🎯 价格目标监控"卡片**
2. **修改目标价格**：2200 → 2300
3. **点击"💾 保存配置"**
4. **看到"✅ 配置已保存"**

### 验证生效

查看监控程序终端，10 秒内会看到：

```
🔄 检测到配置变化，正在应用新配置...

✅ 新配置已应用

💡 行情监控配置：
   价格目标监控: ✅ 开启
      - ETH-USDT: 达到 2300 USDT
```

## 完成！🎉

现在你可以：

- ✅ 随时修改配置，10 秒内自动生效
- ✅ 手机访问，像 App 一样使用
- ✅ 无需重启程序，无需编辑代码

## 🐛 遇到问题？

### 无法访问配置页面

1. 检查 Web 服务器是否启动
2. 检查端口 3000 是否被占用
3. 尝试访问 http://127.0.0.1:3000

### 手机无法访问

1. 确保手机和电脑在同一局域网
2. 检查电脑防火墙是否开放 3000 端口
3. 使用电脑的局域网 IP（不是 localhost）

### 配置修改后没有生效

1. 检查监控程序是否运行
2. 等待最多 10 秒（配置检查间隔）
3. 查看监控程序日志
