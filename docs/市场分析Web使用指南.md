# 📊 市场分析 Web 界面使用指南

## 🎯 功能概述

市场分析功能提供专业的技术分析和开多/开空建议，支持多币种配置。

### 核心特点

- ✅ **使用 API Key 认证**：避免 IP 限制
- ✅ **支持多币种**：ETH、BTC、SOL、DOGE、XRP 等
- ✅ **实时分析**：基于火币真实历史数据
- ✅ **智能建议**：综合 10+ 个技术指标

---

## 🚀 快速开始

### 1. 启动服务

```bash
# 确保已配置 API Key
# .env 文件中需要有：
# HTX_ACCESS_KEY=你的key
# HTX_SECRET_KEY=你的secret

# 启动 Web 服务器
npm run web
```

### 2. 访问页面

打开浏览器访问：http://localhost:3000

点击顶部的 **📊 市场分析** 标签页

---

## 📱 使用步骤

### 步骤 1：选择合约

从下拉菜单中选择要分析的合约：

```
可选合约：
- ETH-USDT
- BTC-USDT
- SOL-USDT
- DOGE-USDT
- XRP-USDT
```

### 步骤 2：输入当前价格

输入合约的当前价格，例如：`2150`

**获取当前价格的方法**：
- 从火币 App 查看
- 从监控程序的实时输出获取
- 从其他行情软件获取

### 步骤 3：输入持仓成本（可选）

如果你有持仓，可以输入持仓成本价，例如：`2100`

这样会额外显示持仓成本分析。

### 步骤 4：开始分析

点击 **🔍 开始分析** 按钮

等待 2-5 秒，系统会：
1. 调用火币 API 获取历史 K线数据
2. 计算技术指标（MA、RSI、MACD、布林带）
3. 分析多时间窗口趋势
4. 生成智能交易建议

## 🔧 技术指标说明

### MA (移动平均线)

- **MA20**：20 周期移动平均
- **MA50**：50 周期移动平均

**用途**：
- 价格 > MA20 > MA50：多头排列，看涨
- 价格 < MA20 < MA50：空头排列，看跌

---

### RSI (相对强弱指标)

- **范围**：0-100
- **< 30**：超卖，可能反弹
- **> 70**：超买，可能回调
- **30-70**：正常区间

---

### MACD

- **金叉**：看涨信号
- **死叉**：看跌信号

---

### 布林带

- **上轨**：阻力位
- **中轨**：均线
- **下轨**：支撑位

**用途**：
- 价格触及上轨：超买，可能回调
- 价格触及下轨：超卖，可能反弹

---

## ⚠️ 注意事项

### 1. 数据延迟

- K线数据有 1 分钟缓存
- 建议使用最新的实时价格

### 2. 信号准确性

- 技术分析不是 100% 准确
- 建议结合基本面分析
- 设置合理的止损

### 3. 风险控制

- 不要满仓操作
- 根据信心指数调整仓位：
  - 70-100%：中等仓位（30-50%）
  - 50-70%：轻仓（10-30%）
  - < 50%：观望
- 严格执行止损

### 4. 市场环境

- **震荡市场**：降低仓位，频繁止损
- **趋势市场**：可适当加仓，顺势而为
- **极端行情**：停止交易，保护本金

## 🔄 更新频率

### 建议更新间隔

- **短线交易**：每 5-15 分钟更新一次
- **中线交易**：每 1-4 小时更新一次
- **长线交易**：每天更新一次

### 何时需要重新分析

- 价格大幅波动（> 3%）
- 重要新闻事件
- 技术形态突破
- 持仓盈亏达到预期

## 💡 使用技巧

### 1. 多币种对比

同时分析多个币种，选择信号最强的：

```
ETH-USDT: 🟢 做多 75%
BTC-USDT: 🟡 观望 45%
SOL-USDT: 🟢 做多 80%  ← 选择这个
```

### 2. 结合持仓成本

如果有持仓，输入成本价可以看到：
- 当前盈亏
- 成本位置（是否在有利位置）
- 是否需要加仓或减仓

### 3. 设置价格提醒

根据分析结果，在配置页面设置价格目标：
- 做多建议：设置上方目标价
- 做空建议：设置下方目标价
- 观望建议：设置突破价格提醒

### 4. 记录交易日志

建议记录每次分析结果和实际操作：
- 分析时间
- 建议操作
- 信心指数
- 实际操作
- 结果反馈

这样可以总结经验，提高判断准确率。

## 🐛 常见问题

### Q: 分析失败怎么办？

A: 检查以下几点：
1. 确保 .env 文件中配置了 API Key
2. 检查网络连接
3. 确认输入的价格格式正确
4. 查看浏览器控制台错误信息

### Q: 为什么有些时间段没有数据？

A: 可能是：
1. 该时间段火币没有数据
2. 网络请求失败
3. 数据缓存过期

### Q: 信心指数一直很低怎么办？

A: 说明当前市场信号不明确，建议：
1. 等待更明确的趋势
2. 降低仓位或观望
3. 关注其他币种

### Q: 可以自动交易吗？

A: 当前版本仅提供分析建议，不支持自动交易。
建议人工判断后再操作，确保风险可控。
